
# Temperature Converter Assignment

A responsive web application for converting temperatures between Celsius and Fahrenheit.

## Features

- **Bidirectional Conversion**: Convert from Celsius to Fahrenheit or Fahrenheit to Celsius
- **Input Validation**: Validates numeric input with user-friendly error messages
- **Responsive Design**: Mobile-friendly interface with professional styling
- **Real-time Calculation**: Instant temperature conversion results

## Usage

1. Enter a temperature value in the input field
2. Select your conversion mode (Celsius to Fahrenheit or vice versa)
3. Click "Calculate Conversion" to see the result

## Conversion Formulas

- **Celsius to Fahrenheit**: `(C × 9/5) + 32`
- **Fahrenheit to Celsius**: `(F - 32) × 5/9`
